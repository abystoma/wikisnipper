# Wikisnipper

A webapp for viewing wikipedia articles using the wikipedia [Search Api](https://www.mediawiki.org/wiki/API:Search)

## Installation

```
git clone https://github.com/abystoma/wikisnipper.git
cd wikisnipper
npm install
npm start
```
